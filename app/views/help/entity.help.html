What is an entity

We need to get user's values from their input. Intent will tell you the action then entities will hold the values of the user's input. Consider following example,
  - I want cheese pizza
  - I want veg pizza
  - I want onion pizza

As you can see all three questions will relate to order pizza, there for we can create an intent called 'order_pizza' But each of these questions contains different values. In order to extract those values, we'll use entities. Let's see how we should create an entity for above case.

  1) Type your entity name (for our example, entity name is pizza)
  2) Type your entity description (for our example, 'This entity will hold the pizza type')
  3) Set your lookup strategies (for our example, trait)
  4) Now you can add a value and expressions, lets add value expression pair for our example,
        1) Value: 'cheese'
            Expressions: 'I want cheese pizza, I want to order cheese pizza, make me a cheese pizza'
        2) Value: 'veg'
            Expressions: 'I want veg pizza'
        3) Value: 'onion'
            Expressions: 'I want onion pizza'
  5) Add entity data (These data will help our bot when user ask these questions)
  6) Create your entity
    Note: You don't need to create entity and value pairs at first, you can always create them later.

You can always go to the App -> Train Bot page and verify your questions
  1)Type your question (for our example, 'I want cheese pizza')
  2)Click on check button
  3)It will show your entity with the value 'cheese' which you just created

Note:
If the entity did not showed to your question, you can simply select 'pizza' as the Entity field and 'cheese' as the value field. Then click on validate and recheck again. now your intent should be there :)


Lookup Strategies:

Trait:
When the entity value is not inferred from a keyword or specific phrase in the sentence. There is no obvious association between certain words in the sentence and the value of the entity, but rather you need the sentence as a whole to determine the value.

Keywords:
When the entity value belongs to a predefined list, and you just need substring matching to look it up in the sentence.
